## Overview

This AI-powered business analyst agent helps teams refine project specifications through collaborative workflows. It enables iterative improvement cycles with automated analysis and human feedback integration, producing structured project documentation for clearer requirements and better project outcomes.

## Core features

1. **Multi-Agent Collaboration Workflow**: Orchestrates a collaborative workflow between analyst and critic agents with human feedback integration for iterative project specification refinement
2. **Iterative Project Refinement**: Supports up to 3 revision cycles with automated analysis and human feedback integration for continuous project improvement
3. **Structured Project Artifact Generation**: Automatically generates structured project specifications with titles, descriptions, goals, and functional requirements using AI analysis
4. **State Management System**: Maintains project state across iterations including descriptions, artifacts, feedback, and revision counts using Pydantic models
5. **Critic Feedback Integration**: Incorporates automated critique feedback to validate and improve project specifications before human review

## Installation

**Prerequisites:** requires Python ^3.10

Install AIST-Business-analyst-agent using one of the following methods:

**Build from source:**

1. Clone the AIST-Business-analyst-agent repository:
```sh
git clone https://github.com/HammonDDDDD/AIST-Business-analyst-agent
```

2. Navigate to the project directory:
```sh
cd AIST-Business-analyst-agent
```

3. Install the project dependencies:
```sh
pip install -r requirements.txt
```
