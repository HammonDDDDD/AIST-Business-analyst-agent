# AIST Business Analyst Agent

## Overview
A collaborative agent system that generates and refines project artifacts through iterative feedback cycles involving AI agents and human input.

## Purpose
To automate business analysis by creating project documentation through a structured workflow where an AI analyst generates drafts, receives critique from both AI and human reviewers, and iteratively improves the output based on feedback. The system supports Telegram integration for user interaction and manages revision cycles with human-in-the-loop participation.
